Ruby ist eine interpretierende Sprache, d.h. es ist keine Datentyp-Festlegung der Variablen nötig.
Der Interpreter erkennt an dem Zugewiesenden Wert um welchen Datentyp es sich Handelt.

	Bsp.
		var = "Ein String"     #var ist eine Objekt der Klasse String
		var.class			   #erzeugt Ausgabe String,also den Datentyp

Besonderheiten von Strings:
	Konkatenieren:
		var = "String"
		puts "Dies ist ein " + var

	Ausgeben von Ruby-Code im String:
		Das Konstukt: " Zeichenkette #{ ruby code } " ermöglicht es in einer Zeichenkette Code auszuführen und Variablen einzufügen
		Bsp:
			movie = "Goonies"
			rank = 10
			puts "#{movie} hat am#{Time.now} das Ranking #{rank}"
